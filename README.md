# TO-DO-LIST-HW-FOR-AMITAVA-SIR

A simple Flask TODO List web application created for learning Flask CRUD operations.

## Features

- Add Task
- View Task
- Edit Task
- Delete Task

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Project Structure

todo_project/
│
├── app.py
├── templates/
│   ├── index.html
│   ├── add_task.html
│   └── edit_task.html
│
└── static/
    └── style.css

## Run Project

Install Flask:

pip install flask

Run the project:

python app.py

Open browser:

http://127.0.0.1:5000
