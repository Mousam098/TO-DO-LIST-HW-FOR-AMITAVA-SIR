# TO-DO LIST WEB APPLICATION

A simple and user-friendly TO-DO List web application built using **Python** and **Flask**. This project allows users to add, manage, and organize daily tasks efficiently through a clean web interface.

---

## 📌 Project Overview

This project was developed as a beginner-friendly Flask application to understand:

- Flask framework basics
- Routing and templates
- Handling user input
- Web application structure
- CRUD operations for tasks

The application helps users keep track of their tasks in a simple and organized way.

---

# 🚀 Features

- ➕ Add new tasks
- 📝 Update/Edit tasks
- ❌ Delete tasks
- 📋 Display all tasks
- 🌐 Simple web interface using Flask
- ⚡ Lightweight and easy to run

---

# ⚙️ Installation & Setup Using UV

## 1️⃣ Clone the Repository

Open CMD and run:

```bash
git clone https://github.com/Mousam098/TO-DO-LIST-HW-FOR-AMITAVA-SIR.git
```

---

## 2️⃣ Go to Project Folder

```bash
cd TO-DO-LIST-HW-FOR-AMITAVA-SIR
```

---

# 📦 Install UV

UV is a fast Python package manager.

## 3️⃣ Install UV

Run this command in CMD:

```bash
pip install uv
```

Check if UV is installed:

```bash
uv --version
```

---

# 🧪 Create Virtual Environment

## 4️⃣ Create Environment Using UV

```bash
uv venv
```

This creates a virtual environment folder named `.venv`

---

## 5️⃣ Activate the Environment

### Windows CMD

```bash
.venv\Scripts\activate
```

After activation, you will see something like:

```bash
(.venv)
```

in the CMD terminal.

---

# 📥 Install Project Dependencies

## 6️⃣ Install Flask

```bash
uv pip install flask
```

OR if your project has `requirements.txt`:

```bash
uv pip install -r requirements.txt
```

---

# ▶️ Run the Flask Application

## 7️⃣ Start the Server

```bash
python app.py
```

OR

```bash
uv run app.py
```

---

# 🌐 Open in Browser

After running the project, you will see:

```bash
http://127.0.0.1:5000
```

Open this link in your browser.
